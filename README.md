# 💱 Currency Converter (React)

A modern, responsive **Currency Converter** application built with **React**, **Tailwind CSS**, and a **custom React hook** for fetching real-time exchange rates.  
The app features a clean glassmorphism UI, currency swapping, and live conversion using a public currency API.

## 🔌 API Used
This project uses the free currency exchange API provided by fawazahmed0/currency-api.

Endpoint: https://cdn.jsdelivr.net/npm/@fawazahmed0/currency-api@latest/v1/currencies

Features: No API key required, fast CDN-based access, and daily updated rates.

## ▶️ Getting Started
Follow these steps to run the project locally.

1️⃣ Clone the Repository
``` Bash
git clone [https://github.com/your-username/currency-converter-react.git](https://github.com/your-username/currency-converter-react.git)

cd currency-converter-react
```
2️⃣ Install Dependencies
``` Bash
npm install
```
3️⃣ Start the Development Server
```Bash
npm run dev
```
